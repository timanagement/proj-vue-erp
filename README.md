# SX

> Admin dashboard.

## Build Setup

```bash
# install dependencies
npm install

# or 
cnpm install

# serve with hot reload at localhost:8088
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
